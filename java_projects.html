Bank Account Management System (Java)

A simple Java application for managing bank accounts with deposit, withdrawal, and balance check functionalities.

public class BankAccount {
    private double balance;
    
    public BankAccount(double initialBalance) {
        this.balance = initialBalance;
    }
    
    public void deposit(double amount) {
        balance += amount;
    }
    
    public void withdraw(double amount) {
        if (amount <= balance) {
            balance -= amount;
        } else {
            System.out.println("Insufficient funds!");
        }
    }
    
    public double getBalance() {
        return balance;
    }
}

Student Grade Calculator (Java)

A Java program to calculate students' grades based on their scores.

import java.util.Scanner;

public class GradeCalculator {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("Enter student score: ");
        int score = scanner.nextInt();
        
        char grade;
        if (score >= 90) grade = 'A';
        else if (score >= 80) grade = 'B';
        else if (score >= 70) grade = 'C';
        else if (score >= 60) grade = 'D';
        else grade = 'F';
        
        System.out.println("The student grade is: " + grade);
        scanner.close();
    }
}
